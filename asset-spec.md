# Asset Specification — Ingredient Illustrations

Every illustration file must meet these requirements so assets drop cleanly into the composition layer.

## Technical Format

| Property | Requirement |
|---|---|
| File format | PNG (preferred) or SVG |
| Background | Transparent |
| Aspect ratio | 1:1 (square) |
| Canvas size | **600 × 600 px** |
| Subject position | Centred, with ~15% margin on all sides (subject fills a ~510×510 area) |
| Subject | Single ingredient, raw and uncooked unless that is not possible (e.g. smoked trout) |
| Scale reference | Consistent perceived visual weight — a single lemon and a single lettuce leaf should feel proportionate when placed side by side on the board |
| Style | Clean editorial illustration. Flat colour with subtle drop shadow or gentle gradient shading. Not photographic. |
| Colour profile | sRGB |
| Naming | Exactly the `assetKey` from the draw list, lowercase, hyphens for spaces, `.png` extension |
| Delivery | Flat directory, one file per ingredient |

## Generic / Fallback Assets (commission once each)

| Filename | Used for |
|---|---|
| `herb-sprig.png` | Any herb not individually illustrated (parsley, mint, dill, thyme, rosemary, etc.) |
| `spice-jar.png` | Dry spices, spice blends, powders (za'atar, dukkah, paprika, cumin, garam masala, etc.) |
| `bottle.png` | Sauces and liquid condiments (soy sauce, fish sauce, vinegars, mirin, chilli oil, etc.) |
| `bottle-wine.png` | Wine and spirits (white wine, red wine, sake, cognac, etc.) |
| `nut-bowl.png` | Nuts and seeds (almond, walnut, pine nut, pepita, sesame, etc.) |
| `leaves.png` | Salad leaves not individually drawn (rocket, kale, bok choy, etc.) |
| `grain.png` | Grains, pasta, rice (farro, quinoa, couscous, brown rice, etc.) |
| `generic-produce.png` | Catch-all for any unlisted ingredient |
